# Issue #001 背景ノードの扱い

## 現状

背景は255件まで登録可能としているが、
画面上でどのように配置・表示するかが未定義。

## 候補

### 案1

背景ノードを最上位として表示する。

```
背景
 ↓
上位要求
```

### 案2

背景ノードを独立して複数表示する。

```
背景1
背景2
背景3

 ↓

上位要求
```

### 案3

背景は画面左上に一覧表示し、
グラフには表示しない。

## 優先度

高

# Issue #002 要求の多重親子関係

## 現状

下位要求は親を1つだけ持つ前提となっている。

しかし実際には

```
要求A
   \
    \
     下位要求
    /
要求B
```

のように
複数要求から参照されるケースがある。

## 検討事項

- 親は1つに限定するか
- 複数親を許可するか

## 優先度

高
